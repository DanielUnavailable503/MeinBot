COMMANDS

These are useful right?

INFO

Returns basic info about bot

USAGE
.info

USERINFO

Returns basic info about user

USAGE
.userinfo @user

REPEAT

Repeats the message

USAGE
.repeat message

POKE

Mentions the @user n-times with delay

USAGE
.poke @user n

RANDOM

Random Number Generator!
Optional argument: <1 6> interval of numbers

USAGE
.rand n1 n2

YES/NO

Generates random YES or NO for you to choose Optional argument: n set n-repeats

USAGE
.yn n

HELP

Returns embeded help message

USAGE
.help

GOOGLE

Google Search

USAGE
.google keyword

CURRENCY

Currency Converter

USAGE
.currency 10 eur usd

WIKI

Wikipedia API
Wikipedia Search

USAGE
.wiki keyword

LYRICS

Search lyrics of any song
(Powered by Genius API)

USAGE
.lyrics "author" name

INSTA

Returns Profile picture of someone's instagram

USAGE
.insta insta_name
